# My Algorithm - 个人算法练习仓库
> 个人算法学习与练习仓库，包含各种算法数据结构的实现和LeetCode题解。

## 📚 项目简介

这是一个系统学习算法和数据结构的个人仓库，旨在通过实践掌握各种算法的核心思想和实现。主要使用C++语言实现，按照算法专题进行分类整理。

## 🗂️ 项目结构
```plaintext
my_algorithm/
├── .vscode/ # VS Code 配置文件
│ └── tasks.json # 构建任务配置
├── my_rand.cpp # 随机数生成工具
├── test01.cpp # 斐波那契类型 DP 代码
├── test02.cpp # 矩阵类型 DP 代码
└── 理论基础.txt # 动态规划学习笔记
```
## 📋 代码文件详细说明

### 1. `test01.cpp` - 斐波那契类型动态规划
**包含以下题目的实现：**

#### 已完成题目：
✅ **509. 斐波那契数** (简单)
- 经典的斐波那契数列问题
- 实现方法：递归、记忆化搜索、动态规划

✅ **70. 爬楼梯** (简单)
- 每次可以爬1或2个台阶
- 到达第n阶的不同方法数

✅ **746. 使用最小花费爬楼梯** (简单)
- 每个台阶有cost值
- 计算爬到顶部的最小花费

✅ **1137. 第 N 个泰波那契数** (简单)
- 泰波那契数列：T0=0, T1=1, T2=1, Tn+3 = Tn + Tn+1 + Tn+2

✅ **198. 打家劫舍** (中等)
- 不能偷相邻房屋
- 求最大可偷金额

✅ **740. 删除并获得点数** (中等)
- 删除nums[i]获得点数，同时删除nums[i]-1和nums[i]+1

### 2. `test02.cpp` - 矩阵类型动态规划
**包含以下题目的实现：**

#### 已完成题目：

## 🚀 快速开始

### 环境要求
- C++ 编译器 (支持C++11或更高版本)
- Git 版本控制

### 编译运行示例
```bash
# 1. 克隆仓库
git clone https://github.com/xirill1/my_algorithm.git
cd my_algorithm

# 2. 编译动态规划相关代码
cd dp
g++ test01.cpp -o test01 && ./test01  # 斐波那契类型
g++ test02.cpp -o test02 && ./test02  # 矩形路径类型

# 3. 编译工具函数
g++ my_rand.cpp -o my_rand && ./my_rand
